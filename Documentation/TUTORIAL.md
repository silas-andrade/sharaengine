# Tutorial: Seu Primeiro Jogo (mínimo)

Este é um tutorial mínimo para começar com Shara Engine (versão inicial/stub):

1. Clone o repositório e instale o Rust (veja `Documentation/SETUP.md`).
2. Execute o exemplo:

```bash
cargo run --example simple
```

O exemplo inicializa o subsistema de scripting (stub) e executa o motor (stub). A ideia é servir de ponto de partida para o tutorial completo de criação de um jogo (Pong, plataforma simples) que será adicionado posteriormente.
